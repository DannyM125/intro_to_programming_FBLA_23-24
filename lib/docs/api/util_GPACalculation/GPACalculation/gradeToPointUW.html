Converts a letter grade to its corresponding point value.

This method takes a letter grade as input and returns its corresponding unweighted point value based on the standard GPA scale.

Implementation

static double gradeToPointUW(String grade) {
  switch (grade) {
    case 'A':
      return 4.0;
    case 'B+':
      return 3.5;
    case 'B':
      return 3.0;
    case 'C+':
      return 2.5;
    case 'C':
      return 2.0;
    case 'D':
      return 1.0;
    default:
      return 0.0;
  }
}
