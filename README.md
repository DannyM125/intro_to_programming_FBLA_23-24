# Background info
This was a project created for the 2024 Intro to Programming FBLA event. After presenting our Flutter application, we were chosen to represent the state of New Jersey at the 2024 National Leadership Conference in Orlando, Florida.
GPA calculator app that uses JSON to save data locally. The app is cross-platform with an intuitive user interface that features 20+ available languages (Built in translator package) and color customization.
